# Getting Started
This project contains 2 applications:
    backend
    front-end


## Run Backend

## Entorno virtual

Dentro de la carpeta backend

```
py -m venv venv
source venv/Scripts/activate
```
## Instalar paquetes

```
pip install -r requirements.txt
```

## Correr el Proyecto

```
py src/app.py
```

## Run Frontend

Dentro de la carpeta frontend

## Instalar paquetes

```
npm install
```

## Correr el Proyecto

```
npm start
```
